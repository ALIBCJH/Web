# Theme Color Scheme

## Light Mode
- **Background**: White (#FFFFFF) / Light Gray (#F9FAFB)
- **Text**: Dark Gray (#1E293B) / Medium Gray (#64748B)
- **Primary Accent**: Purple (#7C3AED)
- **Secondary Accent**: Sky Blue (#0EA5E9)
- **Borders**: Light Gray (#E2E8F0)

## Dark Mode
- **Background**: Deep Navy (#0B0F19) / Darker Navy (#0F172A)
- **Text**: White (#FFFFFF) / Light Gray (rgba(255,255,255,0.7))
- **Primary Accent**: Gold (#C5A059)
- **Highlight**: Bright Gold (#D4AF37)
- **Borders**: White with opacity (rgba(255,255,255,0.1))

## Usage
All components now support both light and dark modes with smooth transitions (300ms duration).
The theme toggle button in the navbar allows users to switch between modes.
The selected theme is saved to localStorage and persists across sessions.
